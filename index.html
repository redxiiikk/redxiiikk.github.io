有限状态机

Posted on 2021-07-21 Edited on 2021-09-09 In 算法 Views: Disqus:

什么是有限状态机？

维基百科上关于有限状态机的定义如下：

有限状态机是表示有限个状态及其和状态之间的转移和动作等行为的数学模型。

​

为什么要使用有限状态机？

在思考这个为题之前我们先来想一想如果不使用有限状态机是如何进行状态之间的转移的？最简单的方式就是 if else 进行判断，如果当前的状态满足了某一个 if 条件，就执行其中定义好的代码。

这样的做法简单易懂，但是随着时间的推移和业务的不断变化，就会导致 if else 数量不断的增多，所带来的的影响就是 BUG 的频繁出现和维护成本的增加；同时 if else 的大量使用会导致陷入逻辑泥潭之中，无法有效的在代码中沉淀业务。

那么针对于上述的描述，有限状态机的好处显而易见：

  1. 增加系统架构的可扩展性；
  2. 降低后期的维护成本；
  3. 可以将业务知识沉淀在代码中（业务知识极其的宝贵）。

Linux 下安装 VSCode

Posted on 2021-06-26 Edited on 2021-09-09 In Linux Views: Disqus:

在国内因为网络的限制在安装一些软件的时候会过于耗时甚至导致软件安装失败， VSCode 就是其中之一，这是因为在安装 VSCode 的时候默认会连接到微软在国外站点上部署的服务器，因此我们可以通过修改地址直接链接国内的 Azure 服务来加速这一过程。
只需要软件包下载的域名替换为 https://vscode.cdn.azure.cn 即可。

暂时只能做到快速下载，无法利用 Linux 的包管理工具进行自动更新
