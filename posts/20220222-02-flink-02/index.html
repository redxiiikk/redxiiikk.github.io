Apache Flink 02: DataStream / DataSet Api

2022-02-22

Flink API 层次结构⌗

  1. SQL Api
  2. Table Api
  3. DataStream / DataSet Api
  4. Stateful Processing Function Api

DataStream / DataSet Api⌗

因为 Apache Flink 是流批一体的处理引擎，并且社区目前也有计划统一 DataStream 与 DataSet 的 Api，因此我们将重点学习 DataStream Api ，至于 DataSet Api 与 DataStream Api 基本类似。

DataStream 作业模式：

  1. Source
  2. Transformation
  3. Sink

Source⌗

  1. Flink 内置的一些支持
  2. Flink 提供的一些 Connector
  3. 自定义 Source

Sink⌗

  1. Flink 内置的一些支持
  2. Flink 提供的一些 Connector
  3. 自定义 Source

Transformation⌗

  1. 单流操作
  2. 多流操作

按照状态进行分类：

  1. 无状态操作
  2. 有状态操作

WordConvert⌗

单流、无状态

  1. flatMap
  2. map
  3. fliter

从 Kafka 的 word topic 中每次读取多行文本，然后将多行文本转换为小写的文本，并且全部小写
