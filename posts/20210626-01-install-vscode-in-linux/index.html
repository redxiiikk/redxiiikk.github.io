Linux 下安装 VSCode

2021-06-26

在国内因为网络的限制在安装一些软件的时候会过于耗时甚至导致软件安装失败， VSCode 就是其中之一，这是因为在安装 VSCode 的时候默认会连接到微软在国外站点上部署的服务器，因此我们可以通过修改地址直接链接国内的 Azure 服务来加速这一过程。 只需要软件包下载的域名替换为 https://vscode.cdn.azure.cn 即可。

暂时只能做到快速下载，无法利用 Linux 的包管理工具进行自动更新
